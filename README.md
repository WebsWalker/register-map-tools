## Sample Projects

### 01 — TI INA226 Register Map

Complete register map documentation for the Texas Instruments INA226
36V 16-Bit I2C Current/Power Monitor (datasheet SBOS547E).

Covers all 10 registers (00h to FFh) with full bitfield tables,
access types, reset values, and a calibration worked example
(CAL = 0.00512 / (Current_LSB × R_shunt)).

**Tools:** Python, WeasyPrint
**Source:** TI INA226 datasheet SBOS547E (public)
